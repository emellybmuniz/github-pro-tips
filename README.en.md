## ğŸ” 8. Code Search (Advanced Search)

GitHub has a powerful code search tool that lets you find exactly what you're looking for:

### Search Qualifiers

* `repo:user/repository` â†’ Limit search to a specific repository.
* `org:organization` â†’ Search across all repositories in an organization.
* `language:language` â†’ Filter by language (e.g., `language:python`).
* `path:path` â†’ Search in a specific path (e.g., `path:src/`).
* `symbol:name` â†’ Search for a symbol/function definition.
* `extension:ext` â†’ Filter by file extension.

### Boolean Operators

* `term1 AND term2` â†’ Both terms must be present.
* `term1 OR term2` â†’ At least one term must be present.
* `term1 NOT term2` â†’ Exclude results with term2.

### Regex Search

* Use `/your-regex/` for regular expression searches.
* Example: `/foo(bar|baz)/` finds "foobar" or "foobaz".

### Tips

* Use quotes to search for exact phrases: `"specific function"`.
* Combine qualifiers: `repo:user/repo language:python path:src/`.
* Access [https://cs.github.com](https://cs.github.com) for the new Code Search interface.
